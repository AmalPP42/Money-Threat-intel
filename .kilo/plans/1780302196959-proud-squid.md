# Threat Intel Monetization Plan

## Current State Analysis
The codebase is a well-structured threat intelligence pipeline that:
- Fetches from 9 free RSS feeds
- Deduplicates via SQLite
- Filters by 22 threat keywords
- Scores severity (critical/high/medium/low)
- Maps to MITRE ATT&CK tactics
- Summarizes via Groq free API (or heuristic fallback)
- Distributes to Discord/Slack webhooks
- Saves JSON reports archived in `docs/`

## Code Improvements (Zero-Cost, High-Value)

### 1. Subscriber Keyword Management System
Create a GitHub-based subscription system using Issues:
- Users create issues with their keywords (e.g., "vendor:Microsoft", "cve:CVE-2025")
- Bot labels and processes subscription requests
- Keywords stored in `subscribers.json` in repo
- Pipeline filters articles per subscriber

### 2. Newsletter Automation
Add automated Substack newsletter formatting:
- Generate HTML-formatted daily reports
- Save to `newsletter/` directory for manual copy-paste or API integration

### 3. LinkedIn/Twitter Auto-posting
Add optional social media posting:
- Use GitHub Actions with `gh` CLI to post highlights
- Or generate markdown ready for manual posting

### 4. Email Delivery via GitHub Actions
Leverage GitHub Actions + Gmail SMTP:
- Send emails to subscribers via SMTP
- Use GitHub secrets for email credentials

## Monetization Strategy: Tier-Based Subscription Service
