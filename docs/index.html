⚙️ How It Works

  • Data Collection — 9 free RSS feeds (The Hacker News, BleepingComputer, Krebs on Security, CISA, etc.)
  • Deduplication — SQLite tracks seen articles; no duplicates
  • Filtering — 22 threat keywords (ransomware, RCE, CVE, APT, etc.)
  • Severity Scoring — Critical / High / Medium / Low heuristic engine
  • MITRE Tagging — Auto-maps to ATT&CK tactics (Initial Access, Execution, Impact...)
  • AI Summary — Llama 3 70B via Groq free API generates analyst-style briefing
  • Distribution — Discord/Slack webhooks + email + Substack copy-paste
